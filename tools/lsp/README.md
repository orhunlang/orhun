# Orhun LSP (Arsiv / Prototip)

Bu klasor **Python tabanli eski MVP prototipini** tutar.

## Resmi hat

Uretim icin resmi LSP hattı C++ calistirici icindedir:

```bash
orhun lsp --stdio
```

- `tools/vscode-orhun`

degil C++ LSP hattina eklenmelidir.
